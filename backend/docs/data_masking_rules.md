# 数据分级与脱敏规则清单

## 数据分级

| 级别 | 说明 | 示例字段 |
|------|------|----------|
| 高敏 | 可直接识别个人/企业，泄露风险高 | 身份证号、银行卡号、手机号、护照、SSN |
| 中敏 | 可间接推断或业务敏感 | 姓名、地址、邮箱、合同金额、账号 |
| 低敏 | 可对外展示或仅作统计 | 合同类型、法域、日期范围、条款分类 |

## 模型输入与输出策略

- **输入**：所有发送给外部 LLM 的合同/用户文本必须先经 DLP 脱敏；高敏超过阈值可拒绝调用。
- **输出**：从模型返回的条款引用、建议文本需再次过脱敏管道后再写入报告或展示。
- **存储**：原文仅加密存储；解析结果、导出报告按策略加密或脱敏后存储。
